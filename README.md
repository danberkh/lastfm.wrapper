# lastfm.wrapper 🎧

Personalized music statistics website using the Last.fm API. Wraps all listening last.fm reports in a "Spotify Wrapped" style interface.

## Features
- **Yearly Timeline**: Browse your listening history from 2010 to current year.
- **Top Stats**: Detailed lists of Top Artists, Tracks, and Albums for each year.
- **Genre Analysis**: See your most listened-to genres (tags) per year.
- **YoY Comparison**: Compare your listening activity with the previous year.
- **Dynamic Backgrounds**: Visual atmosphere changes based on your top artist.
- **Share Mode**: Clean UI toggle perfect for taking screenshots.

## Setup
1. Clone the repository.
2. Get your free Last.fm API key at [last.fm/api](https://www.last.fm/api/account/create).
3. Add your API key and username in `js/config.js` (see `js/config.example.js`).
4. Open `index.html` in your browser.

## Built with
- Vanilla JavaScript
- CSS (Custom Variables, Flexbox, Grid)
- Last.fm API
