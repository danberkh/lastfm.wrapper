# lastfm.wrapper 🎧

Personalized music statistics website using the Last.fm API. Wraps all listening last.fm reports in a "Spotify Wrapped" style interface.

## Features
- **Yearly Timeline**: Browse my listening history from 2010 to current year.
- **Top Stats**: Detailed lists of Top Artists, Tracks, and Albums for each year.
- **Genre Analysis**: See my most listened-to genres (tags) per year.
- **YoY Comparison**: Compare my listening activity with the previous year.
- **Dynamic Backgrounds**: Visual atmosphere changes based on my top artist.
- **Share Mode**: Clean UI toggle perfect for taking screenshots.

## Built with
- Vanilla JavaScript
- CSS (Custom Variables, Flexbox, Grid)
- Last.fm API
